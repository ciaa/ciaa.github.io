Lenguaje C

El lenguaje C es uno de los más utilizados para implementar soluciones en sistemas embebidos y es el que tiene más soporte en el Proyecto CIAA.

Micropython

Es una implementación del lenguaje Python optimizada para correr en computadoras como la CIAA y es muy utilizado para enseñar programacion.

Ladder

Es un lenguaje gráfico principalmente orientado a Controladores Lógicos Programables (PLC), pero que también puede ser usado para programar la CIAA.

CIAABOT

Es una plataforma de robótica educativa que permite programar de manera gráfica y está diseñada para la enseñanza de sistemas embebidos.

Java

Es un lenguaje diseñado para que un mismo programa pueda ser ejecutado en distintos dispositivos. La CIAA puede ser programada en Java.
