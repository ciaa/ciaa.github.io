Instalar CIAABOT

La forma más simple de escribir un programa es mediante "CIAABOT", usando programación gráfica. En este link se explica paso a paso cómo hacerlo.

Utilizar Embedded IDE

Para usuarios con más experiencia que quieran usar código fuente hay muchas opciones, algunas basadas en Eclipse. La más simple es "Embedded IDE".

Aprovechar la sAPI

Para escribir el código en Embedded IDE hay varios lenguajes y bibliotecas disponibles. Para empezar se sugiere usar el Lenguaje C y la biblioteca sAPI.

Implementar el programa

Utilizando CIAABOT, o el Embedded IDE y la biblioteca sAPI, debemos escribir el programa para que la CIAA resuelva nuestro problema.

Cargar el programa en la CIAA

Una vez que tenemos el programa escrito en la PC lo cargamos en la CIAA para que empiece a ejecutarse y así resolver el problema que abordamos.

La CIAA lo empieza a ejecutar

Una vez que la CIAA tiene el programa cargado en su memoria comenzará a ejecutarlo y podremos ver en acción la solución que le hayamos programado.
