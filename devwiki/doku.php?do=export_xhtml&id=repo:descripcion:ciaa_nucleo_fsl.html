CIAA-NUCLEO-FSL

La CIAA-NUCLEO-FSL es una versión minimalista y modular de la CIAA-FSL pensada como módulo de procesamiento para aplicaciones de bajo costo que requieran una conexión Ethernet. Posee conectores de expansión tipo pinera simple para poder agregarle fácilmente electrónica adicional específica de cada aplicación.
