Ponchos CIAA

Llamamos “Ponchos” a los módulos que se conectan sobre algún procesador de la familia CIAA. El nombre “Poncho” se utiliza entre la comunidad del proyecto CIAA como sinónimo de “Shield” o placa de expansión.

Estos módulos extienden la funcionalidad de alguna CIAA, EDU-CIAA, picoCIAA, etc., agregando el hardware necesario para distintas aplicaciones.

Ponchos existentes

Existen varios Ponchos diseñados, que proveen distintas funcionalidades, como por ejemplo Poncho Osciloscopio, Poncho Robot Sumo, Poncho Interfaz de Usuario, Poncho Ethernet, Poncho FPGA, Poncho Biopotenciales, Ponchos educativos, Poncho Celda de Carga, Poncho Detector de Caídas, Poncho Wifi PicoCIAA, Poncho IoT y varios más. El listado completo y detallado de Ponchos puede accederse en este enlace.

¿Cómo empezar?

Si quiere visualizar o editar el diseño de los circuitos impresos y los diagramas esquemáticos de las placas entonces instale en su computadora la herramienta abierta multiplataforma Kicad que es la que se utiliza para diseñar las distintas placas de la familia CIAA.

Luego acceda al repositorio en Github CIAA Ponchos, seleccione “Clone or Download” y “Download ZIP”, descomprima y tendrá una copia local.

En la línea de comandos, para obtener el repositorio localmente es: 

 git clone https://github.com/ciaa/Ponchos
