OpenOCD

OpenOCD (Open On Chip Debugger) es un tool que se puede utilizar para debuggear y ejecutar paso a paso un programa sobre el HW.

La versión de OpenOCD testeada y funcional de la CIAA es la 0.9.0: OpenOCD 0.9.0

Instalación

Instalación de libusb en Ubuntu

Es necesario antes de correr el configure instalar libusb. Esto depende del sistema que se este utilizando. 

sudo apt-get install libusb-1.0-0-dev

Instalacioón de OpenOCD

$ ./configure --enable-maintainer-mode --disable-werror --enable-ftdi 
$ make
$ sudo make install
