Repositorio de ponchos

Se utiliza un repositorio separado al de las placas CIAA para no mezclar ya que son dos flujos separados de trabajo. El repositorio es https://github.com/ciaa/Ponchos

El comando para obtener el repositorio localmente es: 

 git clone https://github.com/ciaa/Ponchos

Cada Poncho tiene su subdirectorio.

Ponchos modelos

Se trata de un conjunto de Ponchos modelo de diferentes tamaños que pueden usarse como TEMPLATES de Kicad, que utilizan bibliotecas que facilitan la creación de un Poncho y con textos instructivos que indican las acciones necesarias para documentar adecuadamente el diseño. Al momento se encuentran disponibles en GitHub los siguientes modelos:

  • Poncho Grande: Sujeción a los cuatro orificios de la EDU-CIAA.
  • Poncho Mediano: Sujeción a la EDU-CIAA con dos orificios.
  • Poncho Chico: Sujeción solo con los dos conectores de la EDU-CIAA.
  • Poncho Izquierdo: Utiliza solamente el conector P2.
  • Poncho Derecho: Utiliza solamente el conector P1.
