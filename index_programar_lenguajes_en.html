C Language

The C Language is the most widely used and established language to implement embedded systems. It's also the most supported language in the CIAA project.

Micropython

It's an implementation of the Python language that's optimized to run on computers similar to the CIAA. It's widely used for teaching programming.

Ladder

It is a graphical language, mainly oriented to PLC (Programmable Logic Controller). But it also can be used to program the CIAA.

CIAABOT

It is an educational robotics platform that allows to program graphically and is designed for the teaching of embedded system, including elementary school.

Java

It is a language designed for developers to write a program once and run it anywhere. The CIAA can run Java programs.
